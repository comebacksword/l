### 题目9(单选):

**查看下列代码 , 选出程序正确的执行结果(  )**

```java
public enum Sex {
    // 每一个枚举项都是枚举的对象
    GIRL("女孩"), BOY("男孩");
    
    private String name;

    private Sex() {
    }
    private Sex(String name) {
        this.name = name;
    }
    public String getName() {
        return name;
    }
}

class Test {
    public static void main(String[] args) {
        System.out.println(Sex.GIRL.getName());
        System.out.println(Sex.BOY.getName());
    }
}
```

#### 选项 :

​	A. 女孩 , 男孩

​	B. null , null

​	C. GIRL , BOY

​	D. 地址 , 地址
